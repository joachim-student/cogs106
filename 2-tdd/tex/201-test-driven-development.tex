  \frametitle{Bayes Factor for Binomial Hypothesis Testing}
  
  \begin{itemize}
    \item The Bayes factor is given by the ratio of the integrals: 
    \[BF = \frac{\int_{0}^{1} p(D|p)p(p|H_1)dp}{\int_{0}^{1} p(D|p)p(p|H_0)dp}\]
    \item This can be computed using numerical integration
  \end{itemize}
